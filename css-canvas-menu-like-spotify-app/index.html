CSS only off-canvas menu like the Spotify app

By Benoît Boucart 06 Apr 2014

I really like the Spotify Android App and his subtle off-canvas menu animation. So I wanted to recreate it in CSS only. Normally it would require Javascript to work, but thanks to the :target CSS pseudo selector it isn’t even needed…

© PicJumbo

The off-canvas layout I made for this tutorial is based on Off Canvas Menu with CSS :target. I made some changes to this code to make it look like the Spotify Android app:

  • A fixed header that doesn't animate
  • A subtle animation on the items on the left and a slide animation

The core concept is based on this CSS code, pay attention to the :target pseudo selector  (will be triggered when you click on a link where #main-nav is targetted (this is SASS syntax):

 1 #main-nav:target {
 2    width: 20%;
 3    a { margin-left: 0; }
 4 }
 5 #main-nav:target ~ .page-wrap {
 6    width: 80%;
 7 }
 8 #main-nav:target ~ .main-header {
 9    .open-menu {
10       display: none;
11    }
12    .close-menu {
13       display: block;
14   }
15 }

This link triggers the :target CSS selector:

1 <a href="#main-navigation">Menu</a>
